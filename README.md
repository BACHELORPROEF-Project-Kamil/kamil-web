# Kamil | Bachelor's Thesis 🛡️

This is the repository for the website of my bachelor's thesis at **Erasmushogeschool Brussel** (Multimedia & Creative Technologies).

**Kamil** is a platform dedicated to increasing digital resilience. Through accessible guides and a smart AI-powered browser extension, Kamil helps users recognize and prevent phishing and online fraud. This repository contains the source code for the web platform.

---

## 🚀 Project Overview
- **Developer:** Arno Baeck
- **Contact:** arno.baeck@student.ehb.be / arno.baeck@live.be
- **Tech Stack:** Next.js 16, Tailwind CSS 4, TypeScript, React 19
- **Status:** Under Development

## ✨ Key Features
- **Educational Guides:** Dynamic content rendered from Markdown files.
- **Responsive Design:** Fully optimized for both mobile and desktop experiences.
- **AI Integration Insights:** Information regarding the linked browser extension that analyzes websites in real-time.
- **Performance:** Built with the latest React and Next.js features for optimal speed.

## 🛠 Installation & Development

### Prerequisites
- Node.js (Version 18 or higher)
- npm or yarn

### Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/kamil-web.git
   cd kamil-web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure
- `app/`: Contains all pages, components, and global styles (Next.js App Router).
- `content/guides/`: Educational articles stored as Markdown files.
- `public/`: Static assets such as images, custom fonts, and SVGs.
- `app/lib/`: Utility functions for data fetching and guide processing.
- `app/components/`: Modular UI components (Navbar, Footer, GuideCard, etc.).
